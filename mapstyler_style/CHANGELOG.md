## 0.1.0

- Initial release.
- Style, Rule, ScaleDenominator.
- Symbolizers: Fill, Line, Mark, Icon, Text, Raster.
- Filters: Comparison, Combination, Negation, Spatial, Distance.
- Expressions: Literal, Function (PropertyGet, ArgsFunction, Case, Step, Interpolate).
- Geometry model: Point, LineString, Polygon, Envelope.
- StyleParser interface with ReadStyleResult / WriteStyleResult.
- Full GeoStyler JSON serialization (fromJson / toJson).
